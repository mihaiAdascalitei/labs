Descrierea jocului :

X si O (engleza tic-tac-toe) este un joc pentru doi jucatori, "X" respectiv "0", care marcheaza pe rand cate o casuta dintr-un tabel cu 3 linii si 3 coloane.
Jucatorul care reuseste primul sa marcheze 3 casute adiacente pe orizontala, verticala sau diagonala castiga jocul.

Fiecare jucator va avea de plasat pe o tabla un simbol. Jucatorii nu pot plasa decat un simbol in aceeasi casuta.
